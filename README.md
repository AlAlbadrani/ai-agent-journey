# AI Agent Journey

A conversational AI agent built with Python and the Claude API.

## Features
- Multi-turn memory — remembers context across the conversation
- Calculator tool — solves math expressions
- Weather tool — fetches city weather (currently simulated)
- Multilingual — responds in the language you write in

## How to run
1. Clone the repo
2. Create a virtual environment and activate it
3. Install dependencies: `pip install anthropic python-dotenv`
4. Add your API key to a `.env` file: `ANTHROPIC_API_KEY=your-key`
5. Run: `python day4.py`
